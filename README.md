**My Personal Code Snippet Library** ‚Äì a private project to store, manage, and quickly search all my code snippets in one place. Built with **React** and **Firebase**, it lets me access snippets from any system while keeping them private.

**How to Use Locally:**  
1. Clone: `git clone https://github.com/poojaroogi/CodeNest.git`

3. Replace `firebase.js` with your Firebase config  
4. Run: `npm start` ‚Üí Open [https://code-nest-xjc3-89v7grxbe-pooja-s-rs-projects.vercel.app/)
